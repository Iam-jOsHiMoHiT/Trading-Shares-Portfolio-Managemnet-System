# Trading-Shares-Portfolio-Managemnet-System
Trading Shares Portfolio Management System (TSPMS) is a C programmed system that imitates a Trading App that allows user to :

  1. Buy, sell shares and analyze purchased shares.
  
  2. To analyze stock performance with the help of performance chart.
  
  3. View Real time calculation of Profit/Loss and Profit/Loss percentage.
  
  4. Secure data for future reference.
