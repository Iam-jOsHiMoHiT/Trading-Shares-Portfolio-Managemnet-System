# Trading Shares Portfolio Management System (TSPMS)

TSPMS is a C-based console application that simulates a real-world trading platform. It allows users to manage a simulated stock portfolio — buying and selling shares, tracking performance, and reviewing profit/loss in real time — all through a simple, secure command-line interface.

## Features

- **Buy, Sell & Analyze Shares**
  Add shares to your portfolio, sell existing holdings, and review a breakdown of everything you currently own.

- **Stock Performance Charts**
  Visualize how your holdings are performing over time with built-in performance charts, making it easier to spot trends at a glance.

- **Real-Time Profit/Loss Calculation**
  Instantly see your Profit/Loss (P/L) in absolute terms and as a percentage, updated as you buy and sell.

- **Secure Data Storage**
  User and portfolio data is stored securely, so your trading history and holdings persist between sessions.

## Getting Started

### Prerequisites

- Windows OS (the project ships as a compiled `.exe`)
- No additional dependencies needed to run the pre-built executable

### Installation

1. Clone or download this repository.
   ```
   git clone https://github.com/Iam-jOsHiMoHiT/Trading-Shares-Portfolio-Managemnet-System.git
   ```
2. Navigate to the project folder.
3. Run `portfolio.exe`.

## Usage

1. Launch `portfolio.exe`.
2. Log in using the demo credentials above.
3. From the main menu, choose to:
   - Buy shares
   - Sell shares
   - View your current portfolio and holdings analysis
   - View performance charts
   - Check real-time Profit/Loss and Profit/Loss %
4. Exit the app — your data is saved automatically for the next session.
